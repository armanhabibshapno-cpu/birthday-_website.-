# 💝 100 Reasons Why I Love You

A beautiful multi-page birthday website created with love.

## 🎂 For My Girlfriend's Birthday

This website contains **100 reasons why I love her**, organized into themed chapters with interactive filtering, smooth navigation, and romantic design.

## 📖 Pages

| Page | Description |
|------|-------------|
| **Home** | Hero landing page with animated background |
| **My Letter** | A heartfelt love letter introducing the journey |
| **Categories** | 5 themed chapters of love |
| **All 100** | Complete list with category filtering |
| **Memories** | Beautiful moments we've shared |
| **Forever** | Closing message with love |

## ✨ Features

- 💌 100 unique reasons converted from "Her" to "Your"
- 🎨 6 beautifully designed pages with romantic gradients
- 🔍 Category filtering (Beautiful Soul, Heart of Gold, The Way You Love Me, Your Magic, Forever)
- 📱 Fully responsive — works on phone and computer
- 💖 Smooth animations and hover effects
- 🔗 Easy navigation between all pages
